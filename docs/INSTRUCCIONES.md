Para poder ejecutar este programa, se deben de seguir los siguientes pasos:

0.	Descargar e instalar los requerimientos y el código del programa:

    a. .NET Core 8.0 o superior (con este link).

    b. Clonar el repositorio con git clone git@github.com:MijailCT/ prog_paralela_proyecto_final e ir hacia el directorio donde se clono.

    c. Instalar los paquetes de las librerías utilizadas en este programa con el comando dotnet add src/ package, en caso de que no se instalen al correr de forma automática:

        i. Spectre.Console

        ii. Microsoft.Data.Sqlite

        iii. xunit

        iv. xunit.runner.visualstudio

        v. Microsoft.NET.Test.Sdk

1.	VERIFICAR que en la terminal se está en el directorio del programa que es prog_paralela_proyecto_final, OJO: no en prog_paralela_proyecto_final/src, sino prog_paralela_proyecto_final.

2.	En la terminal/consola de comandos, correr el programa:
a.	Si se está utilizando Windows, con el comando .\correr.bat.
b.	Si se está en GNU/Linux, correrlo con el comando ./correr.sh.
